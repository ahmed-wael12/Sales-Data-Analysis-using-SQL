# 📊 Sales Data Analysis using SQL

## 📌 Overview
This project focuses on analyzing sales data to extract meaningful insights that support business decision-making.

The analysis covers sales performance, customer behavior, and product trends using advanced SQL techniques.

---

## 🎯 Objectives
- Analyze sales performance over time
- Identify top-performing products and categories
- Discover high-value customers
- Measure business growth (Year-over-Year)
- Segment customers and products

---

## 🗂️ Data Model
The dataset follows a **Star Schema**:

- **fact_sales** → transactional sales data  
- **dim_customers** → customer information  
- **dim_products** → product details  

---

## 🛠️ Tools & Techniques
- SQL
- Joins (Fact & Dimension Tables)
- Aggregations (SUM, COUNT, AVG)
- Window Functions (LAG, SUM OVER, AVG OVER)
- Common Table Expressions (CTEs)
- CASE Statements (Segmentation)
- Ranking & KPI Analysis

---

## 🔍 Project Workflow

### 1. Data Exploration
- Explored database structure using `INFORMATION_SCHEMA`
- Identified key columns and dimensions

### 2. KPI Calculation
- Total Sales
- Total Orders
- Total Customers
- Average Price

### 3. Time Analysis
- Monthly & yearly sales trends
- Running totals & moving averages

### 4. Product Analysis
- Top & bottom performing products
- Revenue by category

### 5. Customer Analysis
- Top customers by revenue
- Customer behavior insights

### 6. YoY Growth Analysis
- Compared product performance across years
- Identified growth and decline trends

### 7. Segmentation
- Product segmentation (based on cost)
- Customer segmentation (VIP, Regular, New)

---

## 📊 Key Insights
- A small percentage of customers contributes to most revenue (VIP segment)
- Certain categories dominate total sales
- Some products show consistent growth, while others decline
- Sales trends highlight business growth patterns

---

## 💼 Business Impact
This analysis helps:
- Improve decision-making using data
- Identify high-value customers
- Optimize product strategy
- Detect underperforming products
